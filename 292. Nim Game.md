# <a href="https://leetcode.com/problems/nim-game/">292. Nim Game</a>

```Java
public class Solution {
    public boolean canWinNim(int n) {
        if(n==4)return false;
        if(n<4)return true;
        if((n-1)%4==0||(n-2)%4==0||(n-3)%4==0){
            return true;
        } else {
            return false;
        }
    }
}
```
